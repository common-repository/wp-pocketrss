== Description ==

With this plugin you can easily make your "favorites" blog page using media stored in [getpocket.com](http://getpocket.com). Plugin use rss chanel providing by getpocket.com, to find information about your rss chanel click click [here](http://help.getpocket.com/customer/portal/articles/482632-can-i-subscribe-to-my-list-via-rss-)

== Installation ==

1. Upload `wp-pocketrss` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place [getpocketrss  url=""] hook to any blog page
